# IMDB Movie Data Analysis (EDA)

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on the IMDB Movie Dataset using Python.  
The goal was to clean, analyze, and extract meaningful insights from real-world movie data.

---

## 📂 Dataset Information
- Dataset: IMDB Movie Data
- Records: 1000 movies
- Features include:
  - Title
  - Genre
  - Director
  - Year
  - Runtime
  - Rating
  - Votes
  - Revenue
  - Metascore

---

## 🛠 Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 🔎 Key Steps Performed

### 1️⃣ Data Exploration
- Checked shape, data types, and summary statistics
- Identified missing values
- Checked for duplicate records

### 2️⃣ Data Cleaning
- Handled missing values
- Removed null rows where necessary
- Checked and validated data consistency

### 3️⃣ Feature Engineering
- Created rating classification column
- Split and analyzed genres
- Extracted insights using string operations

### 4️⃣ Data Analysis
- Top revenue movies
- Highest rated movies
- Year-wise average rating
- Director-wise performance
- Runtime-based filtering
- Genre frequency analysis

### 5️⃣ Data Visualization
- Count plots
- Scatter plots
- Year-wise comparisons

---

## 📊 Key Insights
- Certain years generated higher average revenue.
- Some directors consistently produced higher rated movies.
- Popular genres dominate the dataset.
- Higher runtime does not always guarantee higher rating.
